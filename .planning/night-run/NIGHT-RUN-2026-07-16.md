## Protocols (from tonight's incidents — the full doc is reports/git-safety.md)

1. **Single writer.** Exactly ONE committing agent on this checkout at a time. Verifier+planner
   may overlap only because each commits once, to disjoint paths, at the end. Parallel EXECUTORS
   on one checkout are banned (index races swept staged files into siblings' commits this
   milestone). Parallelism = subagents for read-only work, worktrees (`isolation: "worktree"`) for
   writes, or sequential.
2. **Push cadence.** `git push origin main` at every green phase boundary (workflows are
   path-filtered to apps/email-listener/**, untouched; Vercel deploys web — that is the deploy
   playbook working). Never push with a red suite.
3. **Scoped adds only.** Agents `git add` only paths they own. Never `git add -A`. Never
   `git checkout --` without a backup copy (an executor destroyed its own Task 1 tonight).
4. **Look at it.** Eleven bugs shipped through green suites this milestone; every one was found by
   rendering. Every executor runs `npm run screenshot:review` after visual work and READS the PNGs,
   both themes. `npm run test:geometry` must stay green — never weaken a gate to pass.
5. **Known traps (do not rediscover):** bare `npx playwright test` corrupts the live dev server's
   `.next` (use test:geometry / screenshot:review); `npm run build:local` is safe (targets
   `.next-verify`) — `next-env.d.ts` churn afterward is cosmetic (D-61-02, revert it, don't commit
   it); npm workspaces NOT pnpm; Tailwind v4 purges non-literal classes silently; naive greps of
   built CSS lie; unlayered stylesheets (xyflow) beat every layered override; Radix ScrollArea
   viewport shrink-wraps (`display:table`); `pointer-coarse:touch-target` works only since 61-08.
6. **Heartbeat / no dual writers.** The orchestrating session touches
   `.planning/night-run/HEARTBEAT` (timestamp) each turn. A session started by resume-at MUST
   check it: if younger than 15 min, another orchestrator is alive — exit without writing.
7. **Additional user chats** (if opened from phone): read-only advisors, or their own
