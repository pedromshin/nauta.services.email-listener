## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-11-01 | Information disclosure | IDOR via deep-links / graph reads | mitigate (inherited) | The graph reads only api.knowledge.graph, which is importer-filtered from data in 11-01 (D-12); deep-links route to existing /entities/[id] + /emails/[id] surfaces that enforce their own Phase 9/10 tenant rules.
