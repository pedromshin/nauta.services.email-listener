## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| inspector/layers input -> tRPC -> FastAPI | Role/relationship/confirm/deny writes via the server-side proxy. |
| candidate field value -> rendered | LLM-extracted candidate values rendered in the inspector/tree (untrusted text). |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-09-80 | Tampering (XSS) | candidate value / label rendering | mitigate | Candidate values + labels rendered as React text nodes (auto-escaped); no dangerouslySetInnerHTML in the panels. |
| T-09-81 | Repudiation | accidental destructive deny | mitigate | Origin-aware deny: auto-detected deny gets an Undo toast (3000ms); user-drawn deny is non-destructive to geometry (keeps the box) — "your boxes never disappear; the AI's guesses do." |
| T-09-82 | Information Disclosure | API key in editor client | accept/mitigate | Panels call only tRPC mutations (09-04 proxy); no key in client (grep confirms). |
| T-09-83 | Denial of Service | autofill while model down | mitigate | use-autofill-fields non-optimistic + friendly 6000ms degrade toast; controls disabled while extracting (aria-busy). |
| T-09-SC | Tampering | npm/pip/cargo installs | mitigate | No new npm packages (@nauta/ui/lucide/react-pdf present); legitimacy gate N/A. |
</threat_model>
