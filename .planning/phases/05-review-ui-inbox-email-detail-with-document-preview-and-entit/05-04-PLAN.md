## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-13 | Tampering | overlay label / tooltip content | mitigate | entityTypeLabel and extractedFields are rendered as React text children (auto-escaped); no dangerouslySetInnerHTML in any overlay component |
| T-05-14 | Denial of Service | malformed polygon coordinates | mitigate | polygonToRect (unit-tested in 05-01) collapses degenerate/out-of-range polygons to a finite bounding box; the box guards on `location?.polygon` presence before computing |
| T-05-15 | Information Disclosure | PDF bytes via signed URL | accept | URL is a 60s server-minted Storage signed URL (05-02); react-pdf fetches it client-side; lifetime/scope already constrained at the route handler |
| T-05-16 | Tampering | pdf.js parsing untrusted PDFs | accept | pdf.js sandboxes parsing in a worker; the bytes originate from the same Supabase bucket the pipeline ingested; no native code path; standard library risk accepted |
| T-05-SC | Tampering | npm install (react-pdf, pdfjs-dist) | mitigate | react-pdf (publisher: wojtekmaj) and pdfjs-dist (publisher: mozilla) are established first-party packages on npmjs.com — not [ASSUMED]/[SUS]; no legitimacy checkpoint required |
