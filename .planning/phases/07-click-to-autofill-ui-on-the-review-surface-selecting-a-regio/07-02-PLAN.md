  <action>
    Create `apps/web/src/app/emails/[id]/_components/use-autofill.ts` ("use client").
    Export `AutofillPhase`, `ExtractionResult` types and `useAutofill({ emailId })`.
    State (useState, immutable updates via spread):
    `autofillState`, `extractionResults`, `fieldValues` (all `Record<...>`),
    `pickerOpenFor: string | null`.

    `api.emails.autofillComponent.useMutation` — NO optimistic update:
    - onSuccess(data, variables): read `(data as { data: ExtractionResultRaw }).data`;
      set autofillState[componentId]="reviewing"; store extractionResults[componentId]
      with camelCase mapping (extracted_fields→extractedFields,
      confidence_score→confidenceScore, confidence_breakdown→confidenceBreakdown);
      seed fieldValues[componentId] from extractedFields via
      `Object.fromEntries(Object.entries(extractedFields).map(([k,v]) => [k, String(v ?? "")]))`.
    - onError(_err, variables): set autofillState[componentId]="idle";
      `toast.error("AI autofill is unavailable — model access is pending.", { duration: 6000 })`.
      Do NOT surface `_err`.

    `api.emails.confirmComponent.useMutation` — invalidate, no optimistic:
    - onSuccess(_data, variables): `await utils.emails.detail.invalidate({ id: emailId })`;
      set autofillState[componentId]="confirmed"; `toast.success("Fields confirmed")`.
    - onError(_err, variables): set autofillState[componentId]="reviewing";
      `toast.error("Could not confirm fields. Try again.")`.

    Handlers:
    - openPicker(id): set pickerOpenFor=id, autofillState[id]="picking".
    - closePicker(): if current phase is "picking", set it back to "idle"; pickerOpenFor=null.
    - startAutofill(id, slug): set autofillState[id]="extracting", pickerOpenFor=null,
      then `autofillMutation.mutate({ componentId: id, entityTypeSlug: slug })`.
    - setFieldValue(id, key, value): immutably update fieldValues[id][key].
    - confirmFields(id): compute correctedFields = only keys where
      fieldValues[id][key] !== String(extractionResults[id].extractedFields[key] ?? "");
      if empty → null; set autofillState[id]="confirming";
      `confirmMutation.mutate({ componentId: id, correctedFields })`.
    - discardFields(id): clear extractionResults[id] + fieldValues[id]; set
      autofillState[id]="idle". NO API call (D-16 — server record untouched).
    - reset(id): set autofillState[id]="idle".
    Return all state + handlers as a readonly object.
  </action>
