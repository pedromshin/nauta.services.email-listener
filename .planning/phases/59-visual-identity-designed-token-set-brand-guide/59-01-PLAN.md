D-58-01's base (Direction A) is "calm registry rhythm: flat surfaces, hairline rules, zero

## C. Gotchas found during planning — these WILL bite

1. **`--muted-foreground` must be `--faded`, NOT `--pencil`.** The name-match is a trap:
   `--shade` + `--pencil` computes to **4.23 light / 4.02 dark — BELOW the 4.5 AA floor**, and
   the contrast gate pairs `muted`/`muted-foreground` directly. `--faded` gives 5.11 / 5.89.
   `--pencil` ("muted metadata") is still a first-class token — it is legal on `--shelf` (4.57),
   `--leaf` (4.95) and `--bright` (5.43 light / 4.73 dark), but NOT on `--shade`. That usage
   rule is gated in Task 2 and documented in 59-03.

2. **The gates parse LITERAL oklch strings — `parseOklch` throws on `var(--shelf)`.** Writing
   `--background: var(--shelf);` breaks BOTH gates immediately. Task 2 adds a var()-chain
   resolver; do not work around it by inlining duplicate literals (that destroys the single
   source of truth this mapping exists to create).

3. **`readTokenBlock`'s regex is non-greedy to the first `}`.** Keep `:root` and `.dark` flat —
   any nested brace inside those blocks truncates the parse silently.

4. **`readTokenBlock(css, "@theme")` matches `@theme {`, not `@theme inline {`** (the pattern is
   `@theme\s*\{`). Both gates already rely on this. Do not rename or reorder those blocks.

5. **Dark theme is `.dark`, NOT `:root[data-theme="dark"]`.** The sketch uses `data-theme`; the
   app uses next-themes with `attribute="class"` (verified in `apps/web/src/app/layout.tsx`).
   Translate the sketch's `:root[data-theme="dark"]` block into the existing `.dark` block.
   `prefers-color-scheme` + persistence + the Sun/Moon toggle already work
   (`defaultTheme="system" enableSystem`, `app-sidebar.tsx`) — build no theme UI here.

6. **Verified compositing model for the tier-on-wash gate:** composite the wash over its ground
   in GAMMA-ENCODED sRGB (decode -> blend -> re-encode is wrong; blend the encoded channels,
   then decode for luminance). This reproduces 58-IDENTITY's published 4.59 | 6.72 (conf) and
   4.52 | 6.59 (sugg) EXACTLY. Light `sugg` headroom is **0.02**, not 0.09 — the tightest pair
   in the system.
