| Boundary | Description |
|----------|-------------|
| browser → Next proxy route | client POSTs an edgeId + importerId to promote |
| Next proxy → FastAPI | server-side X-API-Key injection; the two-hop key must never reach the client |
| FastAPI 4xx → client | rejection detail must not leak server internals |
