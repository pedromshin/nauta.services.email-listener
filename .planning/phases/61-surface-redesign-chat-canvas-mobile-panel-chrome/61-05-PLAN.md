Make criterion 2's claim — **zero stock React Flow default styling remaining** — true and, more
importantly, checkable.

Purpose: criterion 2 is the only success criterion in this phase phrased as an absolute, and it is
currently false in a way nobody has named. `@xyflow/react/dist/style.css` is imported wholesale;

both canvases renders the library's dark-navy dot with a white ring, which is stock shadcn-era

`var(--border)` dot grid where the sketch designs a 22px `var(--grid)` one, with an explicit
ban-#12 exception written for it.

So the gate parses the library's OWN stylesheet and asserts every selector that paints something
visible has an override. That is the difference between a claim and a fact.

Output: a complete chrome override, the sketch's board, and a gate that fails on a library upgrade
rather than after a user notices.
