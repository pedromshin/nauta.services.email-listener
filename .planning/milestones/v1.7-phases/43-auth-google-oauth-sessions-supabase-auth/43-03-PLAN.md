| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-43-P3-01 | Spoofing (identity injection) | tRPC procedure input | mitigate | `ctx.user` is set server-side from `getUser()`; procedures use `ctx.user`, never `input.userId`; identity-injection test is the acceptance gate (AUTH-03) |
| T-43-P3-02 | Elevation (unauthenticated access) | `protectedProcedure` | mitigate | Middleware throws `TRPCError UNAUTHORIZED` when `ctx.user` is null; unit-proven |
