## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| npm registry → dev machine | Installing a new package pulls untrusted code into the build |
| public env vars → browser bundle | NEXT_PUBLIC_* values are inlined and shipped to every client |
| unverified cookie → authorization | A parsed session cookie is attacker-influenceable until server-verified |
